## Review Notes Draft

- Sway is a live-event app for real-world performer and DJ rooms.
- Patrons join a real room, tip, and submit paid song requests for performer review.
- A paid request is a paid submission for review, not a guaranteed performance.
- The app must not be submitted as a web-only wrapper. Native utility still needs to be implemented and proven before submission.
- Reviewers should be given a live backend window, a performer test account, a room link or QR, and exact instructions for the patron request flow.

## Apple Submission Evidence Still Required

- working iOS native package and bundle ID

- final review notes
- TestFlight build
- internal tester group and test evidence
